Workflow for tile export
--------------------------------------------------------------------------------------
Tile Studio > GIMP > BMP2Tile > source

Use Tile Studio to draw tiles. Ctrl+U gives you the used colors. make a palette out of 
these with the palette manager. Export as 16 color bitmap 4bpp. 
- check background color
- check use this palette as default and for current tile set
- export 16 color bmp


Use GIMP to make sure that the background/transparant color is moved to the front. 
- open bmp
- make sure the background color is OK, else change it
- billede > tilstand > indekseret farvekonvertering 
    generer optimal palet
    maks. farver = 16    klik konverter
- farver > afbildning >rearrange colormap
   drag the background color into position 0, click OK
- overwrite the file (so we update the bmp)
- close (just kasser ændringer)

Open image in BMP2Tile and enjoy!
